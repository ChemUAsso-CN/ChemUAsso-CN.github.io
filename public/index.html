化学式转LaTeX公式

在下面的输入框中输入化学式（用 _a 表示原子个数），然后点击“转换”按钮。

使用说明

1. 输入化学式时，使用 _a 表示原子个数。例如：

  • 水：H_2O
  • 二氧化碳：CO_2
  • 葡萄糖：C_6H_12O_6

2. 反应箭头用 -> 表示。例如：

  • 2H_2 + O_2 -> 2H_2O

3. 气体符号用 ^ 表示，沉淀符号用 v 表示。例如：

  • 氧气气体：O_2^
  • 沉淀：AgClv

4. 离子用 (+n) 或 (-n) 表示电荷。例如：

  • 钠离子：Na^+
  • 氯离子：Cl^-

5. 反应条件（如催化剂）用 [ ] 表示。例如：

  • 过氧化氢分解：2H_2O_2 -> 2H_2O + O_2^ \ [MnO_2]

6. 点击“转换”按钮后，LaTeX 公式会显示在下方。

7. 复制生成的 LaTeX 公式，粘贴到 LaTeX 编辑器（如 Overleaf）中使用。
