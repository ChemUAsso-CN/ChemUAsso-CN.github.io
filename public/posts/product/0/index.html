网站编辑指南：从零开始

2 minute read

化学联合协会网站编辑指南

本指南将帮助你从零开始参与化学联合协会网站的编辑工作。即使你没有任何编程基础，按照本指南的步骤，也能轻松地添加和修改网站内容。

一、基础环境配置

1. 必要软件安装

在开始之前，我们需要安装以下软件：

1. Git - 版本控制工具
   - 访问 https://git-scm.com/downloads
   - 下载 Windows 版本安装包
   - 运行安装程序，全部选择默认选项
   - 安装完成后重启电脑

2. Hugo - 网站生成工具
   - 访问 https://github.com/gohugoio/hugo/releases
   - 下载 hugo_extended_0.129.0_windows-amd64.zip
   - 解压到 C:\Hugo
   - 添加环境变量：
     * 打开"系统属性" -> "环境变量"
     * 在"系统变量"中找到"Path"
     * 点击"编辑" -> "新建"
     * 添加 C:\Hugo

3. Visual Studio Code - 文本编辑器
   - 访问 https://code.visualstudio.com/
   - 下载并安装
   - 建议安装中文语言包

2. 获取项目文件

完成软件安装后，我们需要获取项目文件：

1. 创建工作目录：
   - 在你的电脑上创建一个文件夹（如 D:\web）
   - 这个文件夹将用于存放网站文件

2. 获取项目文件：
   - 打开命令提示符（按 Win+R，输入 cmd）
   - 输入以下命令：
     cd D:\web
     git clone https://github.com/ChemUAsso/ChemUAsso.github.io.git

二、文章编写规范

1. 文件组织

所有文章都应该放在正确的目录下：

content/
└── posts/
    └── product_making/     # 产物制作类文章
        ├── 1.md
        ├── 2.md
        └── 3.md

2. 文章格式

每篇文章必须包含以下部分：

1. 文件头（Front Matter）：
   ```
   ---
   title: "产物制作 X: 产物名称"
   date: 2025-02-23
   description: "简短描述"
   categories: ["产物制作"]
   tags: ["auth: 你的名字", "产物名称", "试剂"]
   author: "你的名字"
   draft: false
   ---
   ```

2. 文章结构：
   - 反应方程式
   - 实验步骤
   - 注意事项（如果有）

3. 数学公式

文章中的数学公式和化学方程式使用 LaTeX 语法：

1. 行内公式：
   - 使用单个 $ 包围
   - 例如：$E = mc^2$

2. 独立公式：
   - 使用两个 $$ 包围
   - 例如：$$\ce{Fe^{3+} + 3C_2O_4^{2-} -> [Fe(C_2O_4)_3]^{3-}}$$

3. 化学方程式：
   - 使用 \ce{} 命令
   - 例如：$\ce{CuSO4 + 2NaOH -> Cu(OH)2 v + Na2SO4}$

三、编辑和预览

1. 使用 VS Code 编辑 

1. 打开项目：
   - 启动 VS Code
   - 点击"文件" -> "打开文件夹"
   - 选择项目文件夹（D:\web\ChemUAsso.github.io）

2. 创建新文章：
   - 在左侧文件树中找到 content/posts/product_making/
   - 右键 -> "新建文件"
   - 输入文件名（如 4.md）

2. 本地预览

在修改文章后，可以在本地预览效果：

1. 启动预览服务器：
   - 打开命令提示符
   - 进入项目目录：cd D:\web\ChemUAsso.github.io
   - 运行命令：hugo server -D

2. 查看预览：
   - 打开浏览器
   - 访问 http://localhost:1313
   - 修改文件后会自动刷新预览

四、提交更改

1. 使用 Git 提交

完成编辑后，需要提交更改：

1. 在 VS Code 中：
   - 点击左侧源代码管理图标（分支图标）
   - 查看更改的文件
   - 在消息框中输入描述（如"添加新文章：xxx的制备"）
   - 点击"提交"按钮
   - 点击"同步更改"推送到远程仓库

2. 检查发布 

1. 等待自动部署：
   - 提交后会自动触发网站更新
   - 大约 1-2 分钟后生效

2. 访问网站查看更改：
   - 打开 https://chemuasso.github.io
   - 确认更改已经生效

五、注意事项

1. 文件命名 

- 使用有意义的文件名
- 避免使用中文和特殊字符
- 使用连字符（-）代替空格

2. 图片处理 

1. 图片存放：
   - 将图片文件放在 static/img/ 目录下
   - 使用有意义的英文名称

2. 图片引用：
   - 在文章中使用 ![描述](/img/图片名.jpg) 格式
   - 建议压缩图片大小，加快加载速度

3. 常见问题解决 

1. 预览显示错误：
   - 检查文章格式是否正确
   - 确认 Front Matter 格式无误
   - 验证数学公式语法

2. 提交失败：
   - 先拉取最新更改：git pull
   - 解决冲突后再提交

3. 图片不显示：
   - 检查图片路径是否正确
   - 确认图片格式支持（jpg/png/gif）

结语

本指南涵盖了参与化学联合协会网站编辑的主要内容。如果你在使用过程中遇到任何问题，可以：

  1. 查看本文相关章节
  2. 咨询其他编辑者
  3. 通过化联群联系管理员

记住：熟能生巧，多尝试几次就会变得得心应手。
