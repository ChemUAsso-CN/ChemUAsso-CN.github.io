化学式转LaTeX

使用说明

支持以下语法：

  • 下标：使用 _n 表示，如 H_2O
  • 上标：使用 ^n 或 ^[n+] 表示，如 Fe^+ 或 SO_4^[2-]
  • 箭头：使用 -> 表示，如 A -> B
  • 等号：使用 = 表示，如 A = B
  • 气体：使用 ^^ 表示上箭头，vv 表示下箭头
  • 加热：使用 ->[delta] 或 =[delta] 表示加热条件
  • 多个条件：使用 ->[条件1][条件2] 或 =[条件1][条件2] 表示
